Week9
    git branch
            branch คือ การเรียงกันของ commit จนเป็นเส้นยาวๆ และในการใช้งาน git จริงๆ
        จะไม่ได้มีแค่ branch เดียวเท่านั้น git ออกแบบมาให้สร้าง branch แยกออกมาได้หลายเส้น

    git pull
            git pull คือการดึงข้อมูล commit ใหม่ๆมาเก็บไว้ในเครื่อง

    merge branch
            merge branch คือการจับ branch มาคู่กัน ในบางครั้งอาจเกิด conflict ขึ้น
        แต่เนื่องจากการแยก branch มักจะใช้สำหรับกรณีที่เขียนโค้ดคนละส่วนกันอยู่แล้ว 
        ดังนั้นตำแหน่งของโค้ดที่เกิด conflict ก็จะมีเพียงบางจุดเท่านั้น
